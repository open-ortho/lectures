The Basics: The Interfaces

  • Frontend. GUI: the Graphical User Interface
  • Backend. API: Application Programming Interface
  • Integrations happen through files or APIs
  • Frontend interfaces are what users see and interact with directly.
  • Backend APIs are how systems communicate with each other programmatically.
  • Integrations between systems happen either by exchanging files or via APIs.
