Weasis and Horos

  • Weasis: viewer; import via Query/Retrieve and DICOMweb; send via C-STORE
  • Horos: viewer + local DB; can receive images via a Storage SCP listener
  • Need a server (Q/R SCP, MWL SCP): dcm4chee-arc or Orthanc
  • Weasis is primarily a viewer/client. The Weasis docs list import via DICOM Query/Retrieve (C-GET/C-MOVE) and DICOMweb, and export via DICOM C-STORE: https://weasis.org/en/index.html
  • Horos includes a local DICOM listener preference pane ("STORESCP") for receiving incoming DICOM connections (Storage SCP): https://deepwiki.com/horosproject/horos/4.3-network-configuration
  • "Reply to query/retrieve" means acting as a Query/Retrieve SCP (answering C-FIND/C-MOVE). These viewers are not typically used as your Q/R server.
